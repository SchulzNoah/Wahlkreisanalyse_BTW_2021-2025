## Wahlkreisanalyse Bundestagswahlen 2021-2025
Interaktives R-Shiny Dashboard zur Analyse der Bundestagswahlen 2021/2025. Interaktive Karten mit Leaflet, dynamische HTML-Tooltips und automatisierte Datenaufbereitung via Tidyverse. Das Shiny-Dashboard ist gehosted und kann [hier](https://schulznoah.shinyapps.io/Wahlkreisanalyse-BTW-2021-2025/) angeschaut werden.
